This folder contains the matlab code for generating the Figure 1 & 2.

Graph1.m -- This is the main program to generate the Figure 1.

Graph2.m -- This is the main program to generate the Figure 2.

Length_Biased_Weibull_density.m -- This is a subroutine program to generate 
the value of the density function g_{lambda,alpha}(y).

Weibull_density.m -- This is a subroutine program to generate the value of 
the density function f_{lambda,alpha}(y).
